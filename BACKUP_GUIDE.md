## 🌐 Publish와 호스팅의 차이

### Publish (게시) ≠ 호스팅 배포

**Publish 버튼:**
- 프로젝트를 "게시 준비" 상태로 만듦
- 아직 인터넷에 올라간 것은 아님!

**호스팅 배포:**
- 실제로 인터넷에 웹사이트 올리기
- 예: Netlify, Vercel, GitHub Pages

### 지금 상황
❌ Publish만 누름 = 아직 백업 안됨
✅ 필요한 것 = 파일 다운로드 또는 호스팅 배포
